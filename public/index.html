Image Search Abstraction Layer

An API to return image search results from Google

Usage

  • /api/imagesearch/<search term> returns a list json object of 10 images for that serch term

    You can optionally add a ?offset=x parameter to return results from the index x

    Examples:
    /api/imagesearch/free code camp
    /api/imagesearch/free code camp?offest=34

  • /api/latest/imagesearch returns the 10 most recent searches

    Example:
    /api/latest/imagesearch
